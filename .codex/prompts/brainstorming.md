# Prompt: Brainstorming

Su dung TRUOC khi code khi scope chua ro, requirement mo ho.

## Steps

1. **Restate problem** — dien giai lai bang 1-2 cau de confirm hieu dung.
2. **Surface assumptions** — liet ke assumption dang gia dinh.
3. **Generate 2-3 options** — kem tradeoff + recommendation.
4. **Surface edge cases** — empty/loading/error/SSR/mobile/dark/i18n/a11y.
5. **Wait for user choice** — KHONG code den khi user xac nhan huong.

## Anti-pattern

- KHONG nhay vao code ngay khi nhan task mo ho.
- KHONG dua 1 option roi implement luon.
- KHONG tu chot — user phai chon.

## Khi nao KHONG can brainstorm

- Task ro, 1 cach lam hop ly.
- Bug fix co root cause cu the.
- User da chi dinh approach.
